# Bluefin 正式登陸 Arbitrum，致力於提供高效能、安全且易用的去中心化交易體驗

Bluefin 是最新於 Arbitrum 上線的去中心化交易所（DEX），專注於效能、安全性與用戶體驗，目標是將中心化交易所的流暢操作體驗帶入去中心化金融（DeFi）世界。

作為一個基於訂單簿的 DEX，Bluefin 提供流動性充足的市場、高效能 API 及順暢的用戶介面。其願景是為機構、專業交易員及首次進入市場的用戶，提供無縫且安全的全球金融市場接入。

---

## Bluefin V2 即將登陸 Sui

Bluefin 亦將於未來數月內在 Sui 區塊鏈上線。屆時，Bluefin 將帶來過往僅有中心化交易所才能實現的規模與效能。下一代 Bluefin 將具備以下特點：

- **卓越的交易體驗**  
  Bluefin 將為首次交易者帶來熟悉的 Web2 入門體驗，同時滿足專業交易者對速度與安全的高標準需求。

- **高吞吐量與最終性**  
  Sui 平行化整個交易流程（不僅僅是執行階段），讓 Bluefin 有潛力處理與中心化交易所同等規模的交易量。

- **去中心化訂單簿**  
  訂單簿相較於自動做市商（AMM）有多重優勢，使專業與機構交易者能在交易所運用既有基礎設施與策略，加速機構流動性與資金流的進駐。

我們將於未來數月內，分享更多關於如何在 Sui 上實現完全無需許可市場的細節，敬請期待！
