# Bluefin 正式登陸 Arbitrum，致力於提升去中心化交易體驗

Bluefin 是最新上線於 Arbitrum 的去中心化交易所（DEX），專注於性能、安全性與易用性，旨在將中心化交易所的體驗帶入 DeFi 世界。

作為一個基於訂單簿的 DEX，Bluefin 提供流動性市場、高效能 API 以及流暢的用戶體驗。該項目的願景是為機構、專業交易員及初次進入市場的用戶，提供無縫且安全的全球金融市場接入。

---

## Bluefin V2 即將登陸 Sui

Bluefin 亦將於未來數月內在 Sui 主網推出。屆時，Bluefin 將帶來過往僅有中心化交易所才能實現的規模與效能。下一代 Bluefin 將具備以下特點：

- **卓越的交易者體驗**  
  Bluefin 將為初次交易者帶來熟悉的 Web2 入門體驗，同時滿足專業交易者對速度與安全的需求。

- **高吞吐量與最終性**  
  Sui 將整個交易流程（不僅僅是執行）進行平行化處理，使 Bluefin 有潛力達到與中心化交易所同等的交易處理量。

- **去中心化訂單簿**  
  訂單簿相較於自動做市商（AMM）有多項優勢，允許專業及機構交易者在交易所上運用現有基礎設施與交易策略，加速機構流動性與資金流的進場。

我們將於未來數月內，分享更多關於如何在 Sui 上實現完全無需許可的市場！
