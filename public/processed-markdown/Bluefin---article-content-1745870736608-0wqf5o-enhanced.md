# Bluefin 正式登陸 Arbitrum，致力於提供高效能、安全且易用的去中心化交易體驗

Bluefin 是最新於 Arbitrum 上線的去中心化交易所（DEX），專注於效能、安全性與使用者體驗，目標是將中心化交易所的優質體驗帶入 DeFi 世界。

作為一個基於訂單簿的 DEX，Bluefin 提供流動性市場、高效能 API 以及流暢的用戶操作體驗。該專案的願景是為機構、專業交易者及首次進入市場的用戶，提供無縫且安全的全球金融市場接入。

---

## Bluefin V2 即將登陸 Sui

Bluefin 預計於未來幾個月內在 Sui 區塊鏈上推出。屆時，Bluefin 將帶來過去僅有中心化交易所才能實現的規模與效能。下一代 Bluefin 將具備以下特點：

### 卓越的交易者體驗
Bluefin 將為首次交易者提供熟悉的 Web2 入門體驗，同時滿足專業交易者對速度與安全的高標準需求。

### 高吞吐量與最終性
Sui 區塊鏈將整個交易流程（不僅僅是執行階段）進行平行化處理，使 Bluefin 有潛力處理與中心化交易所同等數量的交易。

### 去中心化訂單簿
訂單簿相較於自動做市商（AMM）具有多項優勢，讓專業及機構交易者能夠運用既有基礎設施與交易策略於本交易所，加速機構流動性與資金流的進入。

未來幾個月內，我們將分享更多關於如何在 Sui 上實現完全無需許可的市場！
