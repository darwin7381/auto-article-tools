# Bluefin 正式登陸 Arbitrum，致力於打造高效、安全、易用的去中心化交易體驗

Bluefin 是最新於 Arbitrum 上線的去中心化交易所（DEX），專注於提升效能、安全性與用戶體驗，旨在將中心化交易所的操作體驗帶入 DeFi 世界。

作為一個基於訂單簿的 DEX，Bluefin 提供流動性市場、高效能 API 及流暢的用戶介面。其願景是為機構、專業交易者及首次進入市場的用戶，提供無縫且安全的全球金融市場接入。

---

## Bluefin V2 即將登陸 Sui

Bluefin 亦預計於未來數月內在 Sui 區塊鏈上推出。屆時，Bluefin 將實現過去僅有中心化交易所才能達到的規模與效能。下一代 Bluefin 將帶來：

- **卓越的交易者體驗**  
  Bluefin 將為首次交易者提供熟悉的 Web2 新手引導體驗，同時滿足專業交易者對速度與安全的高標準需求。

- **高吞吐量與最終性**  
  Sui 平行化整個交易流程（不僅僅是執行階段），賦予 Bluefin 處理交易量媲美中心化交易所的潛力。

- **去中心化訂單簿**  
  訂單簿相較於自動化做市商（AMM）具備多項優勢，讓專業及機構交易者能將其現有基礎設施和交易策略應用於本交易所，加速機構流動性與資金流的進入。

我們將於未來數月分享更多關於如何在 Sui 上實現完全無需許可的市場！
