# Bluefin 正式登陸 Arbitrum，致力於提升去中心化交易體驗

Bluefin 是最新於 Arbitrum 上線的去中心化交易所（DEX），專注於性能、安全性與易用性，旨在將中心化交易所的體驗帶入 DeFi 世界。

作為一個基於訂單簿的 DEX，Bluefin 提供流動性市場、高效能 API 以及流暢的用戶體驗。該項目的願景是為機構、專業交易員及首次使用者，提供無縫且安全的全球金融市場接入。

---

## Bluefin V2 即將登陸 Sui

Bluefin 也將於未來數月內在 Sui 上線。在 Sui 上，Bluefin 將提供過去僅有中心化交易所才能實現的規模與效能。下一代 Bluefin 將帶來：

- **卓越的交易者體驗**  
  Bluefin 能為首次交易者提供熟悉的 Web2 新手體驗，同時滿足專業交易者對速度與安全的高標準需求。

- **高吞吐量與最終性**  
  Sui 將整個交易流程（不僅僅是執行）進行並行處理，使 Bluefin 有潛力達到與中心化交易所相同的交易處理能力。

- **去中心化訂單簿**  
  訂單簿相較於 AMM（自動做市商）具有多項優勢，允許專業及機構交易者在交易所上運用既有基礎設施與交易策略，加速機構流動性與資金的進入。

我們將於未來幾個月內，分享更多關於如何在 Sui 上實現完全無許可市場的資訊！
