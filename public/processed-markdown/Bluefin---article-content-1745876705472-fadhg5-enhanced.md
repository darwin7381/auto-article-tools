# Bluefin 正式於 Arbitrum 上線，致力於提供中心化交易所級體驗的去中心化交易所

Bluefin 是最新在 Arbitrum 主網上線的去中心化交易所（DEX），專注於效能、安全性與易用性，致力於將中心化交易所的交易體驗帶入 DeFi 世界。

作為一個以訂單簿為基礎的 DEX，Bluefin 提供流動性充沛的市場、高效能 API 以及流暢的用戶體驗。其願景是為機構、專業交易者以及首次進入市場的用戶，提供無縫且安全的全球金融市場存取服務。

---

## Bluefin V2 即將登陸 Sui

Bluefin 亦預計於未來數月內在 Sui 區塊鏈上推出。屆時，Bluefin 將帶來過去僅有中心化交易所才能實現的規模與效能。下一代 Bluefin 將具備以下特點：

- **卓越的交易者體驗**  
  Bluefin 將為首次交易者提供熟悉的 Web2 入門體驗，同時滿足專業交易者對速度與安全的高標準需求。

- **高吞吐量與最終性**  
  Sui 平行化整個交易流程，不僅僅是執行層，使 Bluefin 有潛力達到與中心化交易所相同的交易處理能力。

- **去中心化訂單簿**  
  訂單簿相較於自動做市商（AMM）具有多項優勢，讓專業與機構交易者能在交易所上運用現有基礎設施與交易策略，加速機構流動性與資金流的進入。
