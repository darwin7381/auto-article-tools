# Bluefin 正式登陸 Arbitrum，致力於打造高效、安全且易用的去中心化交易所

Bluefin 為最新上線於 Arbitrum 的去中心化交易所（DEX），專注於效能、安全性與使用者體驗，目標是將中心化交易所的流暢體驗帶入 DeFi 世界。

作為一個基於訂單簿的 DEX，Bluefin 提供流動性充足的市場、高效能 API 以及順暢的用戶操作體驗。Bluefin 的願景是為機構、專業交易者及首次進入市場的用戶，提供無縫且安全的全球金融市場接入服務。

---

## Bluefin V2 即將登陸 Sui

Bluefin 預計於未來數月內在 Sui 區塊鏈上推出。屆時，Bluefin 將帶來過去僅有中心化交易所才能實現的規模與效能。下一代 Bluefin 將具備以下特色：

- **卓越的交易者體驗**  
  Bluefin 將為首次交易者提供熟悉的 Web2 上手流程，同時滿足專業交易者對速度與安全的高標準需求。

- **高吞吐量與最終性**  
  Sui 能將整個交易流程（不僅僅是執行）進行平行化處理，讓 Bluefin 有潛力達到與中心化交易所同等的交易處理量。

- **去中心化訂單簿**  
  訂單簿相較於自動做市商（AMM）具備多項優勢，讓專業及機構交易者能在 Bluefin 上運用既有基礎設施與交易策略，加速機構流動性與交易量的進入。

我們將於未來數月內，分享更多關於如何在 Sui 上實現完全無需許可市場的細節，敬請期待！
