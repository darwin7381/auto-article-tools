# Bluefin 正式於 Arbitrum 上線，致力於提供高效安全的去中心化交易體驗

Bluefin 是最新於 Arbitrum 主網上線的去中心化交易所（DEX），專注於效能、安全性與易用性，旨在將中心化交易所的體驗帶入去中心化金融（DeFi）領域。

作為一個基於訂單簿的 DEX，Bluefin 提供流動性市場、高效能 API 以及流暢的用戶體驗。該項目願景是為機構、專業交易員及首次進入市場的用戶，提供無縫且安全的全球金融市場接入。

---

## Bluefin V2 即將於 Sui 上線

Bluefin 亦將於未來數月內於 Sui 區塊鏈上推出。屆時，Bluefin 將帶來過往僅有中心化交易所才能實現的規模與效能。下一代 Bluefin 將具備以下特點：

- **卓越的交易者體驗**  
  Bluefin 將為首次交易者提供熟悉的 Web2 入門體驗，同時滿足專業交易者對速度與安全性的需求。

- **高吞吐量與最終性**  
  Sui 平行化整個交易流程（不僅僅是執行），使 Bluefin 有潛力處理與中心化交易所相同數量的交易。

- **去中心化訂單簿**  
  訂單簿相較於自動做市商（AMM）有多項優勢，讓專業及機構交易者能夠運用現有基礎設施與交易策略於本交易所，加速機構流動性與資金流入。

我們將於未來幾個月內，分享更多關於如何在 Sui 上提供完全無需許可的市場！
